# Contributing to GenrAItOR

## Report issues
Open an issue in this repos issues page describing the bug, feature request, or documentation update.  Please include steps to reproduce, expected behavior, and minimal data/examples when possible.

## Pull requests
- Fork the repo and create a branch e.g. `bugfix/my-changes` 
- Open a PR against `main` with a clear title and description of the change.
- Address review comments and keep the PR branch up-to-date with `main`.

***

By contributing, you agree your contributions are under the repository [license](license.md).  For questions about contributions or the project, open an issue or contact maintainers listed in `README.md`.  Thanks!
